\chapter{ASCII 码简表}

下面均用十进制数表示 ASCII 码。

\section{字母}

\begin{enumerate}
\item 大写字母 A \textasciitilde Z：65 \textasciitilde 90；
\item 小写字母 a \textasciitilde z：97 \textasciitilde 122。
\end{enumerate}

\section{数字}

\begin{enumerate}
\item 数字 0 \textasciitilde 9：48 \textasciitilde 57。
\end{enumerate}

\section{部分特殊符号}

\begin{enumerate}
\item 感叹号\quad!：33；
\item 双引号\quad"：34；
\item 单引号\quad'：39；
\item 下划线\quad\_：95。
\end{enumerate}
