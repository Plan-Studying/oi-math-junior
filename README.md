# 信息学竞赛与数学：入门级

本仓库是互联网公开资料《信息学竞赛与数学：入门级》的 LaTeX 源码。

本资料是由“学习者”计划编写组一组编纂，在互联网上以 CC BY-SA 4.0 协议公开发布的一篇信息学竞赛资料。本资料是“学习者”计划的一部分。

本资料的知识点编排参考了《全国青少年信息学奥林匹克系列竞赛大纲（2025年修订版）》。在此对 CCF NOI 竞赛委员会和 NOI 科学委员会一并表示感谢。
